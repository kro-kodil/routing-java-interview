# demo-java
### How to run

build using Java 17.

 - Build the application:
   - Windows: `mvnw.cmd clean install`
   - Unix: `mvnw clean install`
 - Run the application:
   - Windows: `mvnw.cmd spring-boot:run`
   - Unix: `mvnw spring-boot:run`
 - Postman / Insomnia / Command line:
   curl --request GET \
    --url http://localhost:8080/routing/CZE/ITA
